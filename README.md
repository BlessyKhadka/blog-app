# Django Blog Application

This project is a basic **Blog application built using Django**, following *Django for Beginners* by William S. Vincent.

The application allows users to:
- Create blog posts using the Django admin
- View all posts on the homepage
- Click a post title to view its individual detail page
- Use static CSS for basic styling
